#  React Todo App (Drag & Drop + Local Storage)

A modern Todo application built with React (Vite).  
This app allows users to manage tasks with full CRUD functionality, drag-and-drop reordering, filtering, and persistent storage.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as complete / undo
- 🔀 Drag and drop to reorder tasks
- 🔍 Filter tasks (All / Active / Completed)
- 💾 Saves tasks automatically using localStorage
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS3
- HTML5 Drag & Drop API
- localStorage

## ⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/todo-app.git

Go into the project folder:

cd todo-app

Install dependencies:

npm install

Start the development server:

npm run dev
